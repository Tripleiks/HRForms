Azure App Registration einrichten
Einmalige Konfiguration für Entra ID Anbindung – danach ausblendbar
▼
1
Azure Portal öffnen → App Registrierungen Gehe zu portal.azure.com → Microsoft Entra ID → App-Registrierungen → Neue Registrierung
2
App konfigurieren Name: Vorschlagswesen-HR | Kontotyp: Nur Konten in diesem Organisationsverzeichnis
Umleitungs-URI: Plattform = Single-Page Application (SPA)
URI = Pfad zu dieser HTML-Datei (z.B. SharePoint-URL oder http://localhost für lokale Tests)
3
API-Berechtigungen hinzufügen API-Berechtigungen → Berechtigung hinzufügen → Microsoft Graph → Delegiert:
User.Read User.ReadBasic.All
→ Administratorzustimmung erteilen (erfordert Globaler Administrator)
4
IDs in diese Datei eintragen Öffne diese HTML-Datei in einem Texteditor und ersetze im <script>-Block:
DEIN_TENANT_ID → Übersicht der App → Verzeichnis-ID (Mandant)
DEIN_CLIENT_ID → Übersicht der App → Anwendungs-ID (Client)
Danach diese Anleitung via setupGuide.style.display='none' ausblenden.
5
Kein Client-Secret nötig! Da das Formular als SPA (Single-Page Application) registriert ist und ausschließlich delegierte Berechtigungen verwendet, braucht keine geheime Schlüssel hinterlegt zu werden. ✅
